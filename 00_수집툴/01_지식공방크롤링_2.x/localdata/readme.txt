2019/1~ (매월 전체데이터를 다운로드받아 처리)

1. localdata.kr 사이트의 다운로드 메뉴에서(http://localdata.kr/devcenter/dataDown.do?menuNo=20001) 생활->목욕탕/찜질방/사우나 분류의 목욕장업 데이터를 다운로드

2. 엑셀파일에서 영업상태명 필드의 값이 '폐업'인 것들은 삭제하고, 맨 뒤에 'MTYPE'이라는 필드를 추가하고 'MTYPE' 필드의 값은 모든 항목에 대해 '찜질방'으로 지정

3. 엑셀파일을 '텍스트 (탭으로 분리) (*.txt)' 포맷으로 저장한 다음, tab은 '|' 문자로 치환, 그리고 " 문자는 제거 (아래와 같이 됨)
	97|목욕장업|11_44_01_P|3000000|3000000-202-1963-00410|19630613||01|영업/정상|01|영업|||||02 7634343|249.10|110855|서울특별시 종로구 창신동 578-1번지 |서울특별시 종로구 창신길 22 (창신동)|03106|현대옥사우나|20171030151739|I|2018-08-31 23:59:59.0|공동탕업|200904.097306757    |452258.945717944    |공동탕업||||1||||||N|||||||||||N|찜질방

4. 맨 윗줄을 아래의 내용으로 치환
	##번호|개방서비스명|개방서비스ID|개방자치단체코드|관리번호|SINCE|인허가취소일자|영업상태구분코드|영업상태명|상세영업상태코드|상세영업상태명|폐업일자|휴업시작일자|휴업종료일자|재개업일자|TELNUM|AREA|소재지우편번호|ADDR|NEWADDR|도로명우편번호|NAME|최종수정시점|데이터갱신구분|데이터갱신일자|업태구분명|X|Y|위생업태명|건물지상층수|건물지하층수|사용시작지상층|사용끝지상층|사용시작지하층|사용끝지하층|한실수|양실수|욕실수|발한실여부|의자수|조건부허가신고사유|조건부허가시작일자|조건부허가종료일자|건물소유구분명|세탁기수|여성종사자수|남성종사자수|회수건조수|침대수|다중이용업소여부|MTYPE@@TAXBATH

5. 이상의 결과를 utf8로 인코딩한 txt 파일로 저장 ('localdata_bath_utf8.txt'라는 이름으로)

5. 아래의 명령 수행 (또는 aa.bat 파일 수행), 결과로 '50603_찜질방_localdata.txt' 파일이 생성됨
	python ..\general_pp.py localdata_bath_utf8.txt ..\final\50603_찜질방_localdata.txt ansi ppname > log.txt
